# ly的测试游戏 Web 版

这是一个独立的 HTML + Canvas + JavaScript 版本。把整个 `web-game` 文件夹发给别人即可，不需要 Xcode，也不需要安装依赖。

## 文件结构

```text
web-game/
  index.html
  styles.css
  game.js
  assets/
    player.png
    player2.png
```

## 打开方式

最简单的方式：

1. 打开 `web-game` 文件夹。
2. 双击 `index.html`。
3. 浏览器打开后点击“开始”或“极难模式”。

如果浏览器限制本地文件，也可以用本地服务器打开：

```bash
cd web-game
python3 -m http.server 8765
```

然后在浏览器访问：

```text
http://127.0.0.1:8765/index.html
```

## 操作方式

- 普通模式：玩家一用 `W A S D` 移动。
- 普通模式冲刺：按住 `空格`，底部白色条是体力。
- 极难模式：玩家一用 `W A S D`，玩家二用 `I J K L`。
- 护盾：白色圆环，可以抵挡一次陨石或射线伤害。

## 排行榜

排行榜会保存在当前浏览器的本地存储里。每个人收到文件夹后，记录都是保存在自己的电脑上，不会同步到别人那里。

- 每个难度和模式保留前 3 名。
- 游戏结束后会自动记录为“未命名”。
- 打开排行榜，点击“未命名”可以输入名字，按回车确认。

## 清除记录

首页左下角有一个小的 `×` 按钮，可以打开清除记录窗口。

密码：

```text
Z5027483
```

密码正确后会清空当前浏览器里的排行榜记录。

## 发给别人

不要只发 `index.html`，要发整个 `web-game` 文件夹。推荐压缩成 zip 后发送：

```text
web-game.zip
```

对方解压后打开 `index.html` 就可以玩。
